## 📌 Overview
This level deepens the understanding of **loop-based logic** applied to number manipulation.  
The focus shifts from simple counters to more complex iterative patterns, including number reversal, primality testing, and sequence generation.

**Phase:** 2 – Looping & Iterative Thinking  
**Level:** 2 – Number-based Looping Logic  
**Language:** Java

---

## 🧠 Concepts Covered
- Number extraction (`%` and `/` operators)
- Reversing numbers
- Palindrome & Armstrong logic
- Prime number algorithms
- Fibonacci sequence generation
- Nested loops (for range-based problems)

---

## ✅ Problems Solved

1. Count the **number of digits** in a given number
2. Print the **reverse** of a given number
3. Check if a number is a **palindrome**
4. Find the **sum of digits** of a number
5. Check if a number is an **Armstrong number**
6. Check if a number is a **perfect number**
7. Print all **prime numbers between 1 and 100**
8. Check if a number is **prime or not**
9. Print **Fibonacci series** up to n terms
10. Print **sum of first n terms** of Fibonacci series
